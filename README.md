# Martha's

## Instalação

Instalando nvm:
```
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
```

Instalando node & npm:
```
nvm install node
```

Instalando express & ejs:
```
npm install express ejs express-ejs-layouts
```

## Execução

Executando o server:
```
node server.js
```
